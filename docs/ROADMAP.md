# Jeopardy ES - Roadmap

## Objetivo

Jeopardy ES es un juego de preguntas y respuestas inspirado en Jeopardy, desarrollado completamente con HTML, CSS y JavaScript Vanilla, con una arquitectura MVC y orientado a facilitar la creación de bancos de preguntas reutilizables.

---

# Estado actual

Versión actual:

v0.1.0 (MVP)

## Funcionalidades implementadas

- Arquitectura MVC
- Configuración centralizada
- Modelos de dominio
- Carga desde JSON
- Tablero dinámico
- Modal de preguntas
- Preguntas reales
- Preguntas usadas
- GitHub Pages
- Diseño responsive básico
